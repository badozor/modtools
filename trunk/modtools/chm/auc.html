auc(modtools)R Documentation

Area Under Curves

Description

This function compute the area under curves.

Usage

auc(x,y)

Arguments

x a numeric vector
y a numeric vector

Value

The function returns a numeric value corresponding to the area under the curves.

Examples

# y <- rpois(1:10)
# d1 <- density(y)
# auc(d1$x,d1$y)
